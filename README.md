# blog-site - Django Project

### Key features of Django used:

* **models** -> To declare model of author, post, comment, tag
* **sqlite3** (inbuilt) -> database
* **templates** -> HTML UI
* **class based views** (function based views are implemented but commented)
* **forms** -> Comment form
